### 리눅스 커맨드라인 기초
```
# print working directory
pwd
```

```
# change directory
cd
```

```
ls // list
```

- ls 뒤에는 -a나 -l 옵션을 붙일 수 있음


### vim 에디터 커맨드
```
# 입력 모드 진입
i 

# 명령 모드 전환
esc
```

```
# 파일 저장 후 vim 에디터 빠져나오기
:wq

# 저장하지 않고 vim 에디터 빠져나오기
:q
```

### 버전관리를 위한 git 커밋
```
#스테이지 영역에 코드 파일 추가
git add

# 커밋
git commit
```
